# Competitor Analysis Skill

## 触发条件
- 新产品立项需要了解市场格局
- 需要评估竞品的功能和市场策略
- 需要制定差异化产品定位

## 执行步骤
1. 确定竞品范围（直接/间接竞品）
2. 收集竞品信息（官网、评测、用户评价）
3. 建立功能对比矩阵
4. 进行 SWOT 分析
5. 输出竞品分析报告和行动建议

## 最佳实践
- 关注竞品的用户评价而非仅看功能列表
- 分析竞品的商业模式和定价策略
- 识别竞品的弱点作为差异化机会
- 定期更新竞品分析

## 常见陷阱
- 过度关注功能对比而忽略体验层面
- 盲目模仿竞品而丢失自身定位
- 忽视行业外的潜在颠覆者
