# Data Analysis Skill

## 触发条件
- 需要评估功能上线效果
- 产品指标异常需要分析原因
- 制定数据驱动的产品决策

## 执行步骤
1. 确定关键指标（北极星指标、AARRR 模型）
2. 建立数据采集和看板
3. 提出假设并设计验证方案
4. 分析数据并得出结论
5. 输出数据驱动的行动建议

## 最佳实践
- 区分虚荣指标与可执行指标
- 先有假设再分析数据
- 关注趋势而非单点数据
- 结合定量与定性数据

## 常见陷阱
- 数据噪声中寻找规律
- 忽视辛普森悖论等统计陷阱
- 只看总量不看构成
