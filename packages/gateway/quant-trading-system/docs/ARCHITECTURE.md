# 量化交易系统架构设计文档

## 📋 系统概览

```
┌─────────────────────────────────────────────────────────────────┐
│                      量化交易系统 (Quant Trading System)          │
├─────────────────────────────────────────────────────────────────┤
│  ┌──────────────┐    ┌──────────────┐    ┌──────────────┐       │
│  │   前端界面    │◄──►│   后端引擎    │◄──►│  数据分析     │       │
│  │  (Frontend)  │    │  (Backend)   │    │  (Analytics) │       │
│  └──────────────┘    └──────────────┘    └──────────────┘       │
│         │                   │                   │               │
│         ▼                   ▼                   ▼               │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                    共享数据层                             │   │
│  │  WebSocket │ REST API │ Message Queue │ Time-Series DB  │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
```

## 🎯 模块职责划分

### 1. 前端交易界面 (Frontend) - 前端工程师负责
**技术栈**: React 18 + TypeScript + Tailwind CSS + Recharts + Zustand

### 2. 后端交易引擎 (Backend) - 后端工程师负责
**技术栈**: Node.js + WebSocket + Redis + PostgreSQL

### 3. 数据分析策略 (Analytics/Strategy) - 量化PM负责
**技术栈**: Python + Pandas + NumPy + Backtrader
