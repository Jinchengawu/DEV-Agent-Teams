## 一、产品概述

### 1.1 产品愿景

打造一套 **智能化、系统化、多维度** 的股票基本面深度尽调分析平台，帮助投资者、研究员、基金经理等专业用户对上市公司进行全面、深入的基本面分析，覆盖财务分析、业务分析、估值分析、风险评估、行业对标等核心维度，输出结构化尽调报告，辅助投资决策。

### 1.2 目标用户

| 用户角色 | 描述 | 核心诉求 |
|---------|------|---------|
| 🧑‍💼 专业研究员 | 券商/基金研究员 | 高效生成深度研报，自动提取关键数据 |
| 💼 基金经理 | 公募/私募基金经理 | 快速横向对比标的，辅助投资决策 |
| 📈 个人投资者 | 具备一定财务知识的散户 | 系统化学习尽调方法论，降低分析门槛 |
| 🏢 PE/VC 投资人 | 一级市场投资人 | 评估标的公司质地，识别风险点 |
| 📰 财经媒体人 | 财经记者/自媒体 | 快速获取公司核心数据，辅助内容创作 |

### 1.3 产品定位

```
┌─────────────────────────────────────────────────────────┐
│                   产品定位矩阵                           │
├──────────────┬──────────────────────────────────────────┤
│   工具定位    │  深度分析工具（非行情终端/非交易系统）       │
│   分析深度    │  深度尽调级别（非浅层财务筛选）              │
│   覆盖范围    │  A股为主，逐步覆盖港股、美股               │
│   交付形式    │  SaaS Web 平台 + API 服务                 │
│   竞争差异    │  AI 驱动的结构化深度尽调 + 自动化报告       │
└──────────────┴──────────────────────────────────────────┘
```

#### 3.1.2 核心财务指标体系

**需求描述**: 系统自动计算并展示全方位的财务指标，覆盖盈利、成长、营运、偿债、现金流五大维度。

**指标体系**:

```
                        ┌─ 毛利率 (Gross Margin)
                        ├─ 净利率 (Net Margin)
            ┌─ 盈利能力 ─┤
            │           ├─ ROE (净资产收益率)
            │           ├─ ROA (总资产收益率)
            │           └─ ROIC (投入资本回报率)
            │
            │           ┌─ 营收增长率 (YoY / CAGR)
            │           ├─ 净利润增长率 (YoY / CAGR)
            ├─ 成长能力 ─┤
            │           ├─ 扣非净利润增长率
  财务指标 ──┤           ├─ 经营现金流增长率
            │           └─ 自由现金流增长率
            │
            │           ┌─ 总资产周转率
            │           ├─ 存货周转率
            ├─ 营运能力 ─┤
            │           ├─ 应收账款周转率
            │           └─ 应付账款周转率
            │
            │           ┌─ 资产负债率
            │           ├─ 流动比率 / 速动比率
            ├─ 偿债能力 ─┤
            │           ├─ 利息保障倍数
            │           └─ 有息负债率
            │
            │           ┌─ 经营现金流/净利润
            │           ├─ 资本支出/折旧摊销
            └─ 现金质量 ─┤
                        ├─ 自由现金流 (FCF)
                        ├─ 现金分红率
                        └─ 现金转化周期 (CCC)
```

**功能清单**:

| 功能项 | 描述 | 优先级 |
|-------|------|-------|
| 指标仪表盘 | 可视化展示五大维度指标，雷达图总览 | P0 |
| 趋势分析 | 任一指标近10年趋势线，标注行业均值 | P0 |
| 行业分位 | 各指标在行业中的排名百分位 | P0 |
| 杜邦分析 | ROE 拆解为 净利率 × 资产周转率 × 权益乘数，瀑布图展示 | P0 |
| 异常标记 | 自动标红异常波动指标（偏离行业均值 >2σ） | P1 |

**需求描述**: 提供全方位的相对估值分析，包括 PE/PB/PS/PEG/EV-EBITDA 等常用估值倍数。

**功能清单**:

| 估值对比 | 与同行业公司估值倍数横向对比 | P0 |
| 综合估值评分 | 多维度估值综合打分 | P1 |

## 八、用户故事 (User Stories)

### 核心用户故事

```
US-001: 作为研究员，我想查看某公司近10年的财务报表数据，以便了解其财务变化趋势。

US-002: 作为研究员，我想对比两家公司的核心财务指标，以便判断哪家公司质地更好。

US-003: 作为基金经理，我想通过 DCF 模型计算公司的内在价值，以便判断当前股价是否被低估。

US-004: 作为个人投资者，我想看到系统自动检测的公司风险点，以便避免踩雷。

US-005: 作为研究员，我想一键生成某公司的深度尽调报告，以便提高研报撰写效率。

US-006: 作为基金经理，我想查看某公司的护城河评估，以便判断其长期竞争优势。
```
