# 🖥️ 股票基本面深度尽调分析系统 — 前端页面结构文档

> **版本**: v1.0  
> **技术栈**: React 18 + Next.js 14 + TypeScript + Ant Design Pro + ECharts

---

## 一、页面路由总览

```
/                                   ── 首页 (Dashboard)
│
├── /search                         ── 公司搜索
├── /watchlist                      ── 自选股看板
│
├── /company/[code]                 ── 公司总览页
│   ├── /company/[code]/overview    ── 概览
│   ├── /company/[code]/financials  ── 财务分析
│   │   ├── ?tab=income             ── 利润表
│   │   ├── ?tab=balance            ── 资产负债表
│   │   ├── ?tab=cashflow           ── 现金流量表
│   │   └── ?tab=dupont             ── 杜邦分析
│   ├── /company/[code]/business    ── 业务分析
│   │   ├── ?tab=revenue            ── 营收结构
│   │   ├── ?tab=moat               ── 护城河
│   │   └── ?tab=industry           ── 行业分析
│   ├── /company/[code]/valuation   ── 估值分析
│   │   ├── ?tab=relative           ── 相对估值
│   │   └── ?tab=dcf                ── DCF估值
│   ├── /company/[code]/risk        ── 风险评估
│   ├── /company/[code]/management  ── 管理层
│   ├── /company/[code]/announcements ── 公告解读
│   └── /company/[code]/ai          ── AI 洞察
│
├── /compare                        ── 多标的对比
├── /reports                        ── 尽调报告管理
│   ├── /reports                    ── 报告列表
│   └── /reports/[id]               ── 报告详情/预览
│
├── /settings                       ── 设置
│   ├── /settings/profile           ── 个人信息
│   ├── /settings/custom-indicators ── 自定义指标
│   └── /settings/notifications     ── 通知设置
│
└── /auth
    ├── /auth/login                 ── 登录
    └── /auth/register              ── 注册
```

## 六、性能优化策略

| 策略 | 描述 |
|------|------|
| 路由懒加载 | Next.js dynamic import，按页面分割代码 |
| 图表懒加载 | ECharts 按需引入，不在首屏的图表异步加载 |
| 数据虚拟化 | 大表格使用 react-window 虚拟滚动 |
| 图片优化 | Next.js Image 组件，WebP 格式 |
| SSR/SSG | 公司概览页 SSG，动态数据 CSR |
| Service Worker | 离线缓存已访问的财务数据 |
| CDN | 静态资源 CDN 加速 |
